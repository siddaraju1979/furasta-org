================= Furasta.Org SVN Repository ===============]

A brief description of the layout of the repository:

trunk		-	this is the main development area, stable versions are kept
			in tags and branches
branches	-	contains various branches of the trunk
tags		-	contains different releases of the CMS, never updated
			once tagged
plugins		-	contains copies of some of the plugins available
			for the CMS
www		-	contains copies of some of the templates available

tools		-	contains other oddball items, such as netbeans
			project files
images		-	contains uncompressed versions of the images used in
			the system
docs		-	contains phpdocumentor documentation of the development
			branch
tests		-	contains various tests conducted on the development
			branch
