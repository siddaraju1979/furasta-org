===================== Furasta.Org Tests ====================
This directory contains tests for the furasta.org system. Two types of
tests are conducted: unit testing and selenium testing.

PHPUnit is used for unit testing - see http://www.phpunit.de/manual/
To run the tests, PHPUnit must be installed on your system - see
 http://www.phpunit.de/manual/3.6/en/installation.html

Selenium is used for user interface testing - see
http://www.phpunit.de/manual/3.6/en/selenium.html
