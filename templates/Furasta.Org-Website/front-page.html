Page Editing made easy

A clear and precise User Interface makes it easy to create, edit and manipulate content. Furasta.Org uses TinyMCE - a software similar to Microsoft Word which means that you probably already know how to use it!

Download
Furasta.Org v1.0 Beta 2.zip

Trash - never lose a page again

The CMS has a built in Trash function which works much the same as the Recycling Bin on Windows, or Trash on Mac. Once a page is removed it is placed in the Trash and from there it can be deleted. Pages can be restored from the Trash at anytime and re-instated in your website.

Download
Furasta.Org v1.0 Beta 2.zip

Advanced Users, Groups and Permissions

Furasta.Org features advanced Permission management so you can ensure that only who you want to access a certain page can access it. Users have their own unique view in the admin area according to the permissions you have assigned them. However, the administrator always has complete control.
